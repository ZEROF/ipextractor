IPEX DBL

A minimalist dynamic block list service. No plugins no stress.

Features

  • Easy firewall integration
  • OpenSource project on GitHub
  • No ads, no tracking, no nonsense
  • Clean Dynamic IP Blocks created from:
    • Spamhaus
    • BinaryDefense
    • DShield
    • Crowdsec
    • Bruteforce
    • Talos
    • BlocklistDE
    • HoneypotProject
  • No duplicate IP's or subnets.

Getting Started

IPEX DBL URL for your firewall can be found here: RAW Github. For integration read provided documentation.

Instructions for: PfSense , OpenSense and Fortigate

Things to Know

  • Dinamic Block List or DBL is the list with IP's updating every 4h.
  • IPv4 only (for now)
  • Using a block list in conjunction with firewalls is a critical strategy for your network security and effectively combating malicious actors.
  • Open GitHub issue if needed.
