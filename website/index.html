IPEX DBL

A minimalist dynamic block list service. No plugins no stress.

Features

  • Easy firewall integration
  • OpenSource project on GitHub
  • No ads, no tracking, no nonsense
  • Clean Dinamic IP Blocks created from:
    • Spamhaus
    • BinaryDefense
    • DShield
    • Crowdsec
    • Bruteforce
    • Talos
    • BlocklistDE
  • We can update IPEX DBL source in any time in the future.

Things to Know

  • Dinamic Block List or DBL is list with IP's in rotation every 24h.
  • IPv4 only (for now)
  • Using a block list in conjunction with firewalls is a critical strategy for your network security and effectively combating malicious actors.
  • Open GitHub issue if needed.
